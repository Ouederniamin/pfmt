## Chapitre 1 — Cancer du sein

### Informations générales
- **Numéro** : 1
- **Titre** : Cancer du sein
- **Description** : Épidémiologie, dépistage, classification TNM, stratégies thérapeutiques et suivi du cancer du sein.
- **URL vidéo YouTube** : `https://www.youtube.com/watch?v=X0B3GhXEbmA`

### Fiche résumé (Markdown)

```
## Points clés — Cancer du sein

### Épidémiologie
- 1er cancer féminin en Tunisie et dans le monde
- Incidence en augmentation, surtout chez les 40-60 ans
- Taux de survie à 5 ans > 85% si dépistage précoce

### Facteurs de risque
- **Non modifiables** : âge > 50 ans, ATCD familiaux (BRCA1/BRCA2), ménarche précoce, ménopause tardive
- **Modifiables** : obésité, alcool, THS prolongé, nulliparité

### Classification TNM
| Stade | T | N | M | Survie 5 ans |
|-------|---|---|---|-------------|
| I     | T1 | N0 | M0 | 98% |
| II    | T2 | N0-1 | M0 | 85% |
| III   | T3 | N1-2 | M0 | 60% |
| IV    | Tout T | Tout N | M1 | 25% |

### Dépistage
- Mammographie bilatérale tous les 2 ans dès 50 ans
- Auto-examen mensuel
- IRM mammaire si haut risque (BRCA positif)

### Traitement
1. **Chirurgie** : tumorectomie + curage axillaire ou mastectomie
2. **Radiothérapie** : systématique après chirurgie conservatrice
3. **Chimiothérapie** : néoadjuvante ou adjuvante selon stade
4. **Hormonothérapie** : tamoxifène (pré-ménopause), inhibiteurs aromatase (post-ménopause)
5. **Thérapies ciblées** : trastuzumab si HER2+
```

### Conclusion

```
Le cancer du sein reste le cancer le plus fréquent chez la femme. Le dépistage précoce par mammographie permet une prise en charge à un stade curable. La stratégie thérapeutique est multimodale, incluant chirurgie, radiothérapie, chimiothérapie et hormonothérapie selon le profil tumoral. La classification moléculaire (Luminal A/B, HER2+, Triple négatif) guide désormais les décisions thérapeutiques.
```

  - D) T-DM1 (Kadcyla®) en cas de résistance ✅
  - E) Inhibiteurs de PARP ❌ (*Indiqués dans les tumeurs BRCA mutées, pas spécifiquement HER2+*)

#### Question 5
- **Énoncé** : Quelles sont les indications de la chimiothérapie néoadjuvante dans le cancer du sein ?
- **Options** :
  - A) Tumeur localement avancée (T3-T4) ✅
  - B) Tumeur inflammatoire ✅
  - C) Cancer du sein T1N0 luminal A ❌ (*Bon pronostic, pas d'indication de chimiothérapie néoadjuvante*)
  - D) Objectif de permettre une chirurgie conservatrice ✅
  - E) Cancer triple négatif pour évaluer la chimiosensibilité ✅

## Chapitre 2 — Cancer du col utérin

### Informations générales
- **Numéro** : 2
- **Titre** : Cancer du col utérin
- **Description** : Rôle du HPV, vaccination, frottis cervical, conisation et protocoles de traitement du cancer du col utérin.
- **URL vidéo YouTube** : `https://www.youtube.com/watch?v=jDKmqS_KpEA`

### Fiche résumé (Markdown)

```
## Points clés — Cancer du col utérin

### Épidémiologie
- 4ème cancer féminin mondial
- Pic d'incidence : 45-55 ans
- Cause principale : Human Papillomavirus (HPV), surtout types 16 et 18

### Prévention
- **Primaire** : vaccination anti-HPV (Gardasil 9® — 9 types dont 16, 18, 31, 33, 45, 52, 58)
- **Secondaire** : frottis cervico-vaginal (FCV) tous les 3 ans de 25 à 65 ans, test HPV après 30 ans

### Classification FIGO (2018)
| Stade | Description | Traitement |
|-------|------------|-----------|
| IA | Micro-invasif (< 5mm profondeur) | Conisation ou hystérectomie simple |
| IB | Limité au col | Chirurgie radicale ± radiothérapie |
| II | Extension paramétriale | Radio-chimiothérapie concomitante |
| III | Paroi pelvienne / hydronéphrose | Radio-chimiothérapie |
| IV | Organes adjacents / métastases | Palliatif ± chimiothérapie |

### Traitement standard
- Stades précoces (IA-IB1) : chirurgie (hystérectomie radicale + curage pelvien)
- Stades avancés (IB2-IV) : radio-chimiothérapie concomitante (cisplatine hebdomadaire)
- Curiethérapie utéro-vaginale en complément
```

### Conclusion

```
Le cancer du col utérin est le seul cancer évitable grâce à la vaccination anti-HPV et au dépistage par frottis. En Tunisie, le diagnostic reste souvent tardif. La radio-chimiothérapie concomitante à base de cisplatine est le standard pour les stades localement avancés. Le suivi repose sur l'examen clinique et l'imagerie.
```

## Chapitre 3 — Cancer de l'endomètre

### Informations générales
- **Numéro** : 3
- **Titre** : Cancer de l'endomètre
- **Description** : Facteurs de risque, biopsie endométriale, stadification FIGO et prise en charge du cancer de l'endomètre.
- **URL vidéo YouTube** : `https://www.youtube.com/watch?v=c1VRppeLisA`

### Fiche résumé (Markdown)

```
## Points clés — Cancer de l'endomètre

### Épidémiologie
- Cancer gynécologique le plus fréquent dans les pays développés
- Âge moyen au diagnostic : 63 ans
- 90% des cas sont des adénocarcinomes endométrioïdes

### Facteurs de risque (hyperœstrogénie)
- Obésité (RR × 3-10)
- Diabète de type 2
- Traitement par tamoxifène
- Syndrome de Lynch (HNPCC)
- Nulliparité
- Ménopause tardive
- THS par œstrogènes seuls

### Diagnostic
- **Signe d'appel** : métrorragie post-ménopausique (90% des cas)
- **Confirmation** : biopsie endométriale (pipelle de Cornier) ou curetage biopsique
- **Bilan d'extension** : IRM pelvienne + scanner TAP

### Classification FIGO
| Stade | Description |
|-------|-----------|
| IA | Limité à l'endomètre ou < 50% du myomètre |
| IB | ≥ 50% du myomètre |
| II | Invasion du stroma cervical |
| III | Extension extra-utérine locorégionale |
| IV | Vessie, rectum ou métastases à distance |

### Traitement
- **Stade I** : hystérectomie totale + annexectomie bilatérale ± curage
- **Stade II** : chirurgie élargie + radiothérapie adjuvante
- **Stade III** : radio-chimiothérapie
- **Stade IV** : chimiothérapie palliative (carboplatine-paclitaxel)
```

## Chapitre 4 — Cancer de l'ovaire

### Informations générales
- **Numéro** : 4
- **Titre** : Cancer de l'ovaire
- **Description** : Marqueurs tumoraux, imagerie, chirurgie de cytoréduction et chimiothérapie dans le cancer de l'ovaire.
- **URL vidéo YouTube** : `https://www.youtube.com/watch?v=s1PP7BXITLE`

### Fiche résumé (Markdown)

```
## Points clés — Cancer de l'ovaire

### Épidémiologie
- 8ème cancer féminin mondial
- Diagnostic souvent tardif (stade III-IV dans 70% des cas)
- Pronostic sombre : survie à 5 ans ~ 45%

### Facteurs de risque
- Mutation BRCA1 (risque 40-60%) / BRCA2 (risque 15-25%)
- Syndrome de Lynch
- Nulliparité
- Endométriose (pour les types endométrioïdes et à cellules claires)

### Facteurs protecteurs
- Contraception orale (↓ 50% du risque après 5 ans)
- Grossesse et allaitement
- Ligature tubaire

### Classification histologique
- **Séreux de haut grade** (70%) — le plus fréquent
- **Endométrioïde** (10%)
- **À cellules claires** (5%)
- **Mucineux** (3%)
- **Séreux de bas grade** (5%)

### Marqueurs tumoraux
- **CA-125** : élevé dans 80% des cancers séreux
- **HE4** : meilleure spécificité que le CA-125
- **Score ROMA** : combine CA-125 + HE4 + statut ménopausique

### Traitement
1. **Chirurgie de cytoréduction** : objectif = résidu tumoral nul (R0)
   - Hystérectomie + annexectomie bilatérale + omentectomie + curage ganglionnaire
2. **Chimiothérapie** : carboplatine + paclitaxel (6 cycles)
3. **Anti-angiogéniques** : bévacizumab en première ligne (stades avancés)
4. **Inhibiteurs de PARP** : olaparib si mutation BRCA
```

## Comment utiliser ces données

1. **Se connecter en tant qu'admin** sur la plateforme
2. **Aller dans Administration → Contenu → Nouveau chapitre**
3. **Étape 1 (Document)** : remplir numéro, titre, description. Uploader un PDF de cours si disponible.
4. **Étape 2 (Contenu)** : 
   - Coller l'URL YouTube dans le champ vidéo
   - Coller la fiche résumé en Markdown
   - Coller la conclusion
5. **Étape 3 (Tests)** : 
   - Ajouter un test, choisir le type (QCM, V/F, Cas clinique)
   - Choisir le placement (Avant/Après le cours)
   - Ajouter les questions avec les options, cocher les bonnes réponses, remplir les justifications
6. **Étape 4 (Finaliser)** : vérifier et enregistrer

### Format des options pour les tests
Chaque question a 5 options (A-E) avec :
- **Texte** de l'option
- **Correct** : coché si bonne réponse
- **Justification** : explication affichée lors de la correction (en italique dans ce fichier)
